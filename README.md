steps to run onlinebanking system project 
1.Download project from GitHub 
2.Install laravel and its prequisities
3.copy project to htdocs folder
4.create database named "example" 
5.navigate inside project through cmd
6.run php artisan migrate to create tables 
7.php artisan serve 
9.open link:http://127.0.0.1:8000 
10.Now register account then login
